Base URL: https://apigee.googleapis.com/v1

Authentication: OAuth 2.0 Bearer Token
Required Scopes: https://www.googleapis.com/auth/cloud-platform

All endpoints follow RESTful conventions:
- GET: Retrieve resources
- POST: Create resources
- PUT: Update resources (full replacement)
- PATCH: Update resources (partial update)
- DELETE: Remove resources

Standard query parameters:
- pageSize: Number of items per page (default varies by resource)
- pageToken: Token for pagination
- filter: Filter expression
- orderBy: Sort field and direction

Standard response format:
{
  "resources": [...],
  "nextPageToken": "string"
}

Error responses follow Google API error format:
{
  "error": {
    "code": 400,
    "message": "Error description",
    "status": "INVALID_ARGUMENT"
  }
}

Rate Limits:
- Standard: 10 requests per second per project
- Burst: Up to 100 requests in 10 seconds
- Quota: Varies by resource type

Retry Strategy:
- Use exponential backoff for 429 (Too Many Requests) and 5xx errors
- Recommended: 1s, 2s, 4s, 8s, 16s intervals
- Maximum retries: 5
